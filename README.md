# oledicons
A library of OLED icons in 0x00 format.

## Overview
oledicons (oli) is an open-source collection of symbols and icons designed for SSD1306 OLED displays.

This project aims to provide a growing set of ready-to-use icons for developers to integrate into menus, user interfaces, and embedded projects. New icons are added over time.

## Motivation
This project started from a lack of available icon resources while developing firmware.

Instead of repeatedly creating icons from scratch, this library was created to make them accessible to others facing the same problem.

## Contributing
Contributions are welcome. You can submit new icons or improvements through pull requests.

## FAQ

Q: How long does it take to create a symbol  
A: It depends on complexity. Detailed icons may take 1 to 2 hours, while simpler ones can take a few minutes

Q: Are these free to use  
A: Yes. The icons are free to use in personal and commercial projects. Attribution is appreciated but not required

## Notes
All icons are created manually, pixel by pixel, which can take a significant amount of time depending on detail
